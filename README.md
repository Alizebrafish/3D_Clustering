# 3D Clustering for Zebrafish Locomotion Analysis

A Windows-based graphical application for unsupervised analysis of 3D zebrafish locomotion-tracking data.

![Workflow overview](figures/workflow_overview.png)

## Overview

This repository provides a ready-to-use application for analyzing 3D locomotion data from zebrafish behavioral experiments. The workflow supports autoencoder-based feature learning, clustering, validation plots, and statistical summaries.

## Key Features

- Graphical user interface for loading 3D tracking files
- Autoencoder-based unsupervised feature extraction
- Multiple clustering algorithms
- Elbow and silhouette analysis
- Dendrogram visualization
- Cluster-level statistical summaries
- CPU and GPU package options

## Installation

### CPU Version

1. Download `CPU_Version.zip`.
2. Extract the ZIP file.
3. Run `start.bat`.
4. Select input Excel files and output folder.
5. Review generated plots and CSV outputs.

### GPU Version

Use only on Windows systems with a compatible NVIDIA CUDA setup.

1. Download `Updated_GPU_Version.tar`.
2. Extract the TAR file.
3. Run `start.bat`.

If the GPU package fails, use the CPU version.

## Input Data

Recommended input structure:

```text
time, x, y, z, group_or_condition
```

## Scientific Interpretation

Cluster labels represent data-driven locomotion states. They should be interpreted using movement-feature summaries, experimental-group enrichment, statistical tests, and biological context.

## Limitations

- Clustering is exploratory and unsupervised.
- Cluster labels do not automatically imply biological causality.
- Results depend on preprocessing, feature quality, and selected parameters.
- GPU execution depends on local hardware and CUDA compatibility.
